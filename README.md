# TP React – Composants et Props

## Description

Ce TP a pour objectif d'apprendre les bases de **React**, notamment :

* la création de composants
* l'utilisation des **props**
* la validation des props avec **PropTypes**
* l'affichage dynamique de listes avec **map()**
